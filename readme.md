# PDF Summarizer

A simple and efficient **PDF Summarizer** built using Python and NLP techniques.
This application extracts text from PDF files and generates concise summaries using a pre-trained model.

---

## Features

* Upload any PDF file
* Extracts text automatically
* Generates meaningful summaries
* Fast and easy to use
* Simple and interactive UI

---

## Tech Stack

* Python
* Streamlit
* Hugging Face Transformers
* PyPDF2

---

## How It Works

1. Upload a PDF file
2. The app extracts text from the document
3. NLP model processes the text
4. A clean summary is generated

---

## Run Locally

```bash
# Clone the repository
git clone https://github.com/your-username/pdf-summarizer.git

# Navigate to project folder
cd pdf-summarizer

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```
